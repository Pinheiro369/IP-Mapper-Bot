# 🌐 **IP Mapper Bot**  
**Bot de rastreamento de IP para Telegram**

# 📌 Como usar:

Basta enviar o comando:

```/ipinfo [IP]```

Exemplo:

```/ipinfo 187.82.292.257```

_O bot retornará os seguintes detalhes:_

```
🌐 Informações do IP
🔢 Endereço IP:
🏷️ Hostname:
🏙️ Cidade:
📍 Região:
🇧🇷 País: BR
📌 Localização:
🏢 Provedor:
🕒 Fuso Horário:
📍 Código Postal:
🌍 URL de Detalhes:
📊 Outras Informações:

🌎 Latitude/Longitude:

🏢 Organização:
```

## 📦 Dependências:

**Certifique-se de ter instalado:**

```
python3  
requests (biblioteca)  
python-telegram-bot (versão 21.10)
```
